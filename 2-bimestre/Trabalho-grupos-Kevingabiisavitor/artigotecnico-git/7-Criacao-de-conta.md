# Como Criar uma Conta no GitHub (Passo a Passo)

O GitHub é uma plataforma online para hospedagem e colaboração em projetos que utilizam Git. A seguir, você aprenderá como criar sua conta de forma simples e rápida.

---

##  1. Acesse o site do GitHub

Abra seu navegador e acesse:

 https://github.com

Na página inicial, clique no botão **"Sign up"** (Cadastrar-se).

---

##  2. Informe seu e-mail

- Digite um endereço de e-mail válido
- Clique em **"Continue"**

---

##  3. Crie uma senha

- Escolha uma senha segura
- Combine letras maiúsculas, minúsculas, números e símbolos
- Clique em **"Continue"**

---

##  4. Escolha um nome de usuário

- Esse será seu identificador público no GitHub
- Deve ser único (o sistema avisará se já estiver em uso)

Clique em **"Continue"** após escolher.

---

##  5. Escolha preferências de comunicação

- O GitHub perguntará se você deseja receber atualizações por e-mail
- Digite **"y"** (sim) ou **"n"** (não)
- Clique em **"Continue"**

---

##  6. Verificação de segurança

- Complete o desafio (captcha)
- ### verificação de e-mail
- Acesse sua caixa de entrada
- Procure o e-mail enviado pelo GitHub
- Clique no link ou insira o código de verificação no site

## Sua conta já está Criada
---

## 7. Configurações iniciais (opcional)

Após criar a conta, você pode:

- Adicionar uma foto de perfil
- Completar seu nome e bio
- Escolher temas (claro/escuro)
- Criar seu primeiro repositório
