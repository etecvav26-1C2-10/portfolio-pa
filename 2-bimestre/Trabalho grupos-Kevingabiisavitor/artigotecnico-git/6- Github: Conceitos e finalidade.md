# 6- github: conceitos e finalidade

## O que é GitHub?

O GitHub é uma plataforma online que utiliza o Git para hospedar repositórios na nuvem. Ele permite armazenar, gerenciar e compartilhar projetos de forma prática, sendo muito utilizado por desenvolvedores ao redor do mundo.

Além de guardar códigos, o GitHub facilita a colaboração entre pessoas em um mesmo projeto.

---

## Conceito de GitHub

Enquanto o Git é uma ferramenta de controle de versão que funciona localmente, o GitHub é um serviço que leva esse controle para a internet.

Com o GitHub, você pode:
- Armazenar seus projetos online
- Acessar seus arquivos de qualquer lugar
- Trabalhar em equipe com outras pessoas
- Compartilhar seu código com o mundo

---

## Para que serve o GitHub?

O GitHub tem várias finalidades importantes no desenvolvimento:

### Armazenamento de projetos
Você pode salvar seus projetos na nuvem com segurança, evitando perdas de arquivos.

### Colaboração em equipe
Várias pessoas podem trabalhar no mesmo projeto ao mesmo tempo, sem conflitos.

### Controle de versões online
Todo o histórico de alterações fica disponível, permitindo acompanhar o desenvolvimento.

### Portfólio profissional
O GitHub também funciona como um portfólio, onde você pode mostrar seus projetos para recrutadores e empresas.

---

## Principais funcionalidades

### Repositórios (Repositories)
São os locais onde os projetos ficam armazenados.

---

### Commits
Registros das alterações feitas no projeto.

---

### Branches
Permitem desenvolver novas funcionalidades sem afetar o projeto principal.

---

### Pull Requests
São pedidos para juntar alterações de uma branch em outra, muito usados em equipes.

---

### Issues
Servem para registrar problemas, ideias ou tarefas dentro do projeto.

---

## GitHub na prática

Um exemplo comum de uso:

1. Você cria um repositório no GitHub  
2. Faz alterações no seu projeto local  
3. Usa o Git para salvar (commit)  
4. Envia para o GitHub com `git push`  
5. Outras pessoas podem ver ou contribuir  

---

## Conclusão

O GitHub é uma ferramenta essencial no mundo da tecnologia, pois facilita o armazenamento, a organização e a colaboração em projetos. Ele complementa o Git e se tornou um padrão no desenvolvimento moderno.
