# portfolio-pa
---
<h1>Bem-vindo ao portifolio-pa</h1>
<p>Este portfólio reúne atividades desenvolvidas utilizando Portugol, com foco no aprendizado de lógica de programação e construção de algoritmos. Ao longo dos exercícios, foram explorados conceitos fundamentais como variáveis, estruturas condicionais, laços de repetição e organização do raciocínio lógico.

Cada atividade apresentada demonstra a evolução no desenvolvimento de soluções computacionais, priorizando clareza, eficiência e boas práticas. O uso do Portugol permitiu compreender de forma mais acessível os princípios da programação, servindo como base para o avanço em linguagens mais complexas.

Este material reflete não apenas o domínio técnico adquirido, mas também o processo de aprendizagem e a capacidade de resolver problemas de maneira estruturada.</p>
